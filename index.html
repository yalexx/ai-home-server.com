AI Home Server FAQ

❓ What is an AI home server?
An AI home server is a dedicated piece of hardware that runs artificial intelligence models locally in your home. Unlike cloud AI services (ChatGPT, Gemini, Claude), an AI home server processes everything on-device — your conversations, documents, and personal data never leave your network. ClawBox is a plug-and-play AI home server built on the NVIDIA Jetson Orin Nano with 67 TOPS of AI performance, pre-loaded with the OpenClaw assistant platform.
❓ Can an AI home server replace my ChatGPT subscription?
For most daily tasks — absolutely. ClawBox runs local language models like Llama 3.1 8B at 15 tokens per second, which handles email management, web research, smart home control, calendar scheduling, document analysis, and general Q&A. For tasks that benefit from frontier-level models (complex coding, advanced reasoning), ClawBox can optionally route queries to cloud APIs — but the choice is always yours, and it's never required.
❓ How much electricity does an AI home server use?
ClawBox draws approximately 15W under typical AI workloads — comparable to a phone charger. Running 24/7 for an entire year, that translates to roughly 130 kWh, or about €25-35 depending on your electricity rate. Compare that to a desktop PC running local AI at 150-300W, which would cost 10-20× more in annual power costs.
❓ Does ClawBox work with Home Assistant?
Yes. ClawBox runs OpenClaw, which integrates with Home Assistant via its REST API and supports MQTT for direct device control. You can manage lights, thermostats, cameras, locks, and automation routines through natural language commands sent via Telegram, WhatsApp, or Discord. No voice assistant hardware needed — your phone is the interface.
❓ What's the difference between an AI home server and a NAS?
A NAS (Network Attached Storage) is optimized for file storage and media serving — think Synology or QNAP. An AI home server like ClawBox is optimized for AI inference, with a dedicated NVIDIA Ampere GPU (1024 CUDA cores) designed to run language models and computer vision tasks. ClawBox includes 512GB NVMe storage for files, but its primary job is running AI workloads, not serving media.
❓ Can I run my own AI models on ClawBox?
Yes. ClawBox supports any GGUF-format model via llama.cpp and Ollama. You can download and run models like Llama 3.1 8B, Phi-3, Mistral 7B, Gemma 2, or any quantized model that fits in 8GB of unified memory. The 512GB NVMe gives you plenty of space to store multiple models and switch between them.
❓ Is 8GB of RAM enough for a home AI server?
For 7-8B parameter models (the sweet spot for local AI in 2026), 8GB of unified LPDDR5 memory is ideal. ClawBox runs Llama 3.1 8B at 15 tokens/second — fast enough for real-time conversation. The key advantage of NVIDIA's unified memory architecture is that both CPU and GPU share the same memory pool with no copying overhead, making every gigabyte count.
